# Animal Restaurant Companion

Finalna wersja aplikacji PWA w React/Vite.

## Pierwsze wdrożenie na GitHub Pages

1. Utwórz nowe, puste repo.
2. Rozpakuj ZIP.
3. Wrzuć CAŁĄ zawartość ZIP-a do repo.
4. Zrób commit na `main`.
5. Wejdź w `Settings → Pages`.
6. W `Source` wybierz `GitHub Actions`.
7. Wejdź w zakładkę `Actions` i poczekaj, aż workflow `Deploy to GitHub Pages` zakończy się zielonym statusem.
8. W `Settings → Pages` pojawi się link do aplikacji.

## Ważne

Nie mieszaj tych plików ze starą wersją aplikacji. To jest kompletna, samodzielna wersja.

## Funkcje

- checklisty: Facilities, Recipes, Customers, Letters, Mementos, Performers, Pets, Fish, Booth Owners
- wishlist
- planner zakupów oparty o ROI
- sync z Animal Restaurant Wiki
- ręczne dodawanie i edycja itemów
- backup/import JSON
- instalacja jako aplikacja na Androidzie
